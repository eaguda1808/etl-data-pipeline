📌 ETL Data Pipeline Project
This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline built using Python and SQL. The pipeline retrieves data from a public API, processes and cleans the data, and stores it in a structured database for further analysis.


🚀 Key Features
Extracts real-time data from an external API
Transforms and cleans data using Python
Loads processed data into a SQL database (SQLite)
Modular pipeline design with separate extract, transform, and load stages
Version-controlled using Git

🛠️ Tech Stack
Python
SQL (SQLite)
Requests (API handling)
Git & GitHub

🎯 Purpose
This project was built to strengthen skills in data processing, backend development, and data pipeline design, while following structured development practices.
